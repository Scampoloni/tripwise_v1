# TripWise – Issue Tracking

Diese Issues wurden aus dem Usability-Test (Dezember 2025) abgeleitet und nach Priorität sortiert. Sie adressieren Layout-Inkonsistenzen, Darkmode-Probleme und UX-Schwächen, die während der Erstnutzung aufgefallen sind. Priorität: **H**och, **M**ittel, **L**ow.

---

## Issues

| ID | Problem (kurz) | Priorität | Betroffene View | Fix-Idee | Status |
|----|----------------|-----------|-----------------|----------|--------|
| **I-01** | Reiseziel-Input weicht optisch von anderen Feldern ab | H | Trip Create | Einheitliche Input-Styles verwenden (Border, Padding, Font) | Todo |
| **I-02** | Autocomplete-Vorschläge im Darkmode kaum sichtbar | H | Trip Create | Kontrast der Dropdown-Liste erhöhen (Background, Text) | Todo |
| **I-03** | Dashboard wirkt im Empty State wenig einladend | M | Dashboard | Onboarding-Hint oder prominenten CTA „Erstelle deinen ersten Trip" ergänzen | Todo |

## Notizen

- **Priorität H (Hoch):** Beeinträchtigt Usability stark, sollte vor nächstem Test behoben sein.
- **Priorität M (Mittel):** Sichtbare UX-Schwäche, aber nutzbar; mittelfristig fixen.
- **Priorität L (Low):** Visuelle Feinheiten, die Qualität verbessern, aber nicht blockieren.
